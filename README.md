# UML SIGAA Atendimento

Projeto acadêmico desenvolvido para a disciplina de UML e Modelagem de Sistemas.

O trabalho apresenta a modelagem completa do sistema SIGAA Atendimento, contemplando:

- Documento de requisitos
- Casos de uso
- Descrições textuais
- Diagrama de classes
- Diagrama de atividades
- Diagrama de estados
- Diagrama de sequência
- Diagrama de componentes
- Diagrama de implantação
- Justificativas de modelagem

Os diagramas foram desenvolvidos utilizando Mermaid e documentados em Markdown.
